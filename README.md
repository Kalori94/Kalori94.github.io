## Ver el sitio localmente

No requiere instalación ni dependencias. Basta con abrir `index.html` en el navegador, o servirlo localmente:

```bash
# Opción 1: abrir directamente
open index.html          # macOS
start index.html         # Windows

# Opción 2: servidor local simple
python3 -m http.server 8000
# luego visita http://localhost:8000
```

## Publicar en GitHub Pages

1. Sube el contenido a un repositorio de GitHub (por ejemplo `kalori94.github.io` para un sitio personal, o cualquier repo con Pages habilitado).
2. En **Settings → Pages**, selecciona la rama `main` y la carpeta raíz (`/`) como fuente.
3. El sitio quedará publicado en `https://<usuario>.github.io/<repo>/`.
